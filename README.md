# Custom-Javascript-Calculator
Javascript Calculator How to Use This Calculator Basic Operations:

Click number buttons (0-9) to enter numbers Click an operator (+, -, ×, ÷) to select your operation Enter the second number Click equals (=) to see your result

Additional Features:

Decimal Point (.) - Add decimals to your numbers Clear (C) - Reset the calculator completely Delete (DEL) - Remove the last digit entered

Keyboard Support:

Type numbers directly using your keyboard Use +, -, *, / keys for operations Press Enter or = to calculate Press Backspace to delete Press Escape or C to clear

Example: To calculate 5 + 3, click: 5 → + → 3 → = (Result: 8) Error Handling: If you try to divide by zero, you'll see an error message that clears automatically after 2 seconds.
